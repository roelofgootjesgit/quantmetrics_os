# Working — inbox & uitvoer

Deze map is **geen** officiële productdocumentatie. Hier hoort werk dat:

- nog geen vaste plek heeft,
- snel geschreven wordt (sessienotities, brainstorm),
- als **todo- of ideeënlijst** dient,
- of als **werk-uitvoerlog** (wat gedaan is, blockers, vervolg).

Gebruik vrije bestandsnamen (`TODO-2026-04.md`, `session-YYYYMMDD.md`, `idee-box.md`, …).  
Verplaats inhoud naar `docs/` of andere repo-docs wanneer het rijp is voor anderen.

**Niet** verplicht om alles te committen; lokaal laten staat ook als je dat prettiger vindt (`.git/info/exclude` of een persoonlijke ignore).

Zie ook: [`QUANTOS_ORCHESTRATION_PLAN.md`](../QUANTOS_ORCHESTRATION_PLAN.md) voor planning op suite-niveau.
